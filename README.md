# 落下回避ゲーム (Falling Dodger)

上から降ってくる障害物（赤いキューブ）を避ける3Dアクションゲームです。
美しく透明感のあるiOS 26風リキッドガラスUIと、3種類のスマホ操作モードに対応しています。

---

## 🌟 特徴
- **リキッドガラスUI**: 超透明で光沢エッジの効いたプレミアムなUIデザイン（日本語対応）。
- **マルチ入力対応**: キーボードだけでなく、スマホで遊べる3つの異なるタッチ操作モードを選択可能。
- **動的な難易度上昇**: 時間経過とともに落下物のスピードと生成頻度が上昇します。
- **デバッグ用無敵機能**: 一時停止（ポーズ）メニューから、テスト用の無敵モードを簡単に切り替え可能。
- **オリジナルBGM**: 自動的にループ再生されるBGMを搭載。

---

## 🚀 開始方法（セットアップ手順）
1. Unityプロジェクトを開きます（動作確認：`Unity 6.5 (6000.5.3f1)`）。
2. Unityエディタの上部メニューバーから **「ゲーム」→「シーンセットアップ」** をクリックします。
   - 必要なプレイヤー、地面、UIキャンバス、イベントシステム、BGM設定などが自動的に生成・配置されます。
3. Unityエディタ中央上部の **「Play（再生）ボタン」** を押してゲームを開始します。

---

## 🎮 操作方法

最初のメニュー画面で、お好みのプレイモードを選択できます。

### 1. キーボードでプレイ（PC）
- **左右移動**: `←` / `→` キー または `A` / `D` キー
- **一時停止**: `Esc` キー または `P` キー
- **リトライ**: `R` キー（ゲームオーバー時）

### 2. スマホ（ボタン操作）
- 画面左右に表示される「◀」「▶」の半透明ボタンをタッチ・長押ししてプレイヤーを左右に移動させます。

### 3. スマホ（スティック操作）
- 画面中央下に表示されるジョイスティック（💀マークつまみ）をドラッグして、直感的に任意の速さでプレイヤーを左右に移動させます。

### 4. スマホ（ドラッグ操作）
- 画面の下半分を左右にスワイプ（ドラッグ）すると、指の動きに合わせてプレイヤーが滑らかに左右に追従します。

## 📂 スクリプト構成
- **[PlayerController.cs](Assets/Scripts/PlayerController.cs)**: プレイヤーの移動制御、移動限界、死亡・無敵判定。
- **[FallingObject.cs](Assets/Scripts/FallingObject.cs)**: 落下物の挙動（下方向への移動）、プレイヤーとの衝突検知。
- **[Spawner.cs](Assets/Scripts/Spawner.cs)**: 落下物の自動生成、時間経過による落下物のスピード・頻度の難易度調整。
- **[GameManager.cs](Assets/Scripts/GameManager.cs)**: スコア・ハイスコア記録、ゲームオーバー、一時停止、各種プレイモードUIの切り替え。
- **[VirtualJoystick.cs](Assets/Scripts/VirtualJoystick.cs)**: スマホのバーチャルジョイスティック操作の入力検出。
- **[DragInputPanel.cs](Assets/Scripts/DragInputPanel.cs)**: スマホのドラッグ（スワイプ）操作の入力検出。
- **[GlassUI.cs](Assets/Scripts/GlassUI.cs)**: リキッドガラスUIに必要な光沢・角丸テクスチャの動的描画生成。
- **[GameSetup.cs](Assets/Scripts/Editor/GameSetup.cs)** (Editor専用): シーンをクリーンアップし、オブジェクトやUI、イベントシステム、BGMを一発で自動構築するエディタ拡張スクリプト。
